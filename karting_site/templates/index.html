Introduction

Karting App project was made to create and record races of go-kart competition at karting.ua. It can record individual laps, give statistic and analyze karting time To navigate throughtout the site you can use navigation panel at the top or read instructions below

Navigation explanation

To receive information about last created race, navigate to or click the link: /recorder #Recorder
To receive the list of all races, navigate to or click the link: /recorder/races #Recorder

Note: This section needs race_id for acess, and it could be acessed only with already existing ID
Links to this resources could be found in the #Recorder part of the project

  • To receive records of the particular race, navigate to /recorder/races/race_id/view #Recorder
  • To receive analyze of karts in the particular race, navigate to /analyzer/race_id #Analyzer
  • To receive static of the particular race, navigate to /analyzer/statistic/race_id #Analyzer

To get details of the API section, navigate to or click the link: /api

Project also have an Admin part, that could create races and edit other race related details: /admin
