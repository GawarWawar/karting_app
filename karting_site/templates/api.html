This is api page for Karting App project
Get back to main page

Project has two api branches:

  • recorder - this branch contains information about individual races and laps of the particular race
    • /api/recorder/list_of_all_races - this endpoint contains information about list of all races
    • /api/recorder/race_records/last_race - this endpoint contains information about last created race
    • /api/recorder/race_records/race_id - this endpoint contains information about list of all races
  • analyzer - this brach contains processed information about pilots, karts and predictions about karts` temp
    • /api/analyzer/analyze_race/race_id - this endpoint contains predictions for the karts that are taking part in the race
    • /api/recorder/race_statistic/race_id - this endpoint contains actual information about pilots, karts and their time

Note: All links marked with Bald, needs race_id for acess
